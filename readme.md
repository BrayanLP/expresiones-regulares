# Curso de Expresiones Regulares

## 1.- Conjunto de Caracteres 

```
    [0-9], [a-z], [A-Z]
```

## 2.- Grupo de Caracteres
```
/\d/, /\w/, /\s/, /\D/, /\W/, /\S/, /./
```

## 3.- Negar Expresi√≥n
```
    /[^expresion]/
```

## 4.- Repetir Patrones

```
    /\d+/ , /\d*/
```

## 5.- Patrones opcionales
```
    /hola m?u?n?d?o?/, /pollo|res|pescado/
```

## 6.- Conteo de patrones
```
    /{1,2}/, /{2,}/
```

## 7.- Patrones sensibles
```
    /\d/i
```

## 8.- Agrupar Patrones
```
    /(aaa)/
```

## 9.- Exec y Match
```
    .exec, .match
```

## 10.- Limite de palabras
```
    /\bHola/
``` 
## 11.- Patronaes al inicio y Fin
```
    /^Inicio/, /Final$/
```
